    KJSencha Module 0.1.1

    Sencha Ext-JS / Sencha Touch Integration Module for Zend Framework 2.

    The project is still a work-in-progress and it is recommended to checkout the Example application if you are interested in using the module

    Installation

    You can install this module via composer by running the following command in your application's root directory:

    $ ./composer.phar require kablau-joustra/kj-sencha

    You can also install this module without composer, assuming you meet all the dependencies specified in composer.json.
